# 任务拆解 - 模块 UI 设计

## 任务概述

基于 `doc/macos-design-spec-v1.md`，对每个功能模块进行详细的 UI 设计方案。每个模块产出一份设计规范文档。

## 设计步骤

每个模块按以下流程：

1. **分析现有组件**：当前模块有哪些组件，布局结构是怎样的
2. **对照 macOS 规范**：应用色彩 token、字体层级、间距网格、圆角阴影规范
3. **产出设计文档**：每个模块一份 markdown 设计规范

## 设计文档模板

每个模块设计文档包含：

1. 组件树（现有 + 设计稿）
2. 布局草图（ASCII art）
3. 色彩应用（哪些 token 用在哪些元素）
4. 字体层级（标题/正文/辅助对应 Apple type scale）
5. 间距方案（各部分间距值）
6. 圆角/阴影（哪一层阴影、多大圆角）
7. 空状态设计
8. 交互细节（hover/active/transition）
