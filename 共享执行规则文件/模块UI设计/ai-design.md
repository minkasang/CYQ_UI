# AI 模块 — UI 设计规范

> 参考：`doc/macos-design-spec-v1.md` · module-inventory

---

## 1. 设计目标

AI 模块合并「AI 聊天」和「AI 总结」两个功能。设计原则：
- **对话式界面**：熟悉的消息气泡布局
- **模型选择清晰**：顶部下拉切换模型和提供商
- **总结功能并入**：作为对话中的一个操作

## 3. 消息气泡规格

### 3.1 用户消息（右对齐）

| 属性 | 值 |
|------|-----|
| 对齐 | `self-end`, 最大宽 70% |
| 背景 | `#0A84FF`（Action Blue） |
| 文字色 | `#ffffff` |
| 圆角 | 12px（右上 4px） |
| 内边距 | 10px 14px |
| 字体 | 13px Body |

### 3.2 AI 消息（左对齐）

| 属性 | 值 |
|------|-----|
| 对齐 | `self-start`, 最大宽 70% |
| 背景 | `rgba(255,255,255,0.08)` |
| 文字色 | `rgba(255,255,255,0.9)` |
| 圆角 | 12px（左上 4px） |
| 内边距 | 10px 14px |
| 字体 | 13px Body |

### 3.3 思考过程（折叠）

```
┌──────────────────────────────────┐
│ 🧠 思考过程              ▼      │ ← 11px Caption
├──────────────────────────────────┤
│ 用户问的是xxx，我需要...         │ ← 11px, text-white/50
└──────────────────────────────────┘
```

## 7. 总结功能入口

在输入框左侧有 `📝 总结` 按钮，点击弹出日记选择器：

```
┌────────────────────────────┐
│ 选择日记总结                │
│ ────────────────────────── │
│ ● 今天 (7月18日)           │
│ ● 昨天 (7月17日)           │
│ ○ 自定义文本...            │
│                    [总结]  │
└────────────────────────────┘
```
