# 待办模块 — UI 设计规范

> 参考：`doc/macos-design-spec-v1.md` · module-inventory · apple-design skill

---

## 1. 设计目标

待办是最高频使用的模块。设计原则：
- **快速浏览**：一眼看到今天要做什么
- **低摩擦操作**：完成/新建都不需要弹窗
- **清晰优先级**：颜色编码一目了然

## 4. 待办项规格

### 4.1 尺寸

| 属性 | 值 |
|------|-----|
| 行高 | 44px（标准列表行） |
| 复选框 | 18×18px，圆角 4px |
| 左边距 | 12px |
| 文字与复选框间距 | 10px |

### 4.2 三行布局

```
┌────────────────────────────────────────────┐
│ ○  买 groceries              🔴 高优先级   │ ← L1: 标题 + 优先级标签
│    今天 14:00 · 🏷 生活 · 🏷 购物          │ ← L2: 元数据行
│    📎 2/3 子任务                            │ ← L3: 子任务进度（可选）
└────────────────────────────────────────────┘
```

- L1：标题（13px Body Bold）+ 优先级标签（11px Caption）
- L2：截止日期 + 标签（11px Caption, text-white/40）
- L3：子任务进度条（有子任务时显示）

### 4.3 完成态

```
┌────────────────────────────────────────────┐
│ ●  完成报告                  🟢 低优先级   │
│    7月17日 · 已完成                        │
└────────────────────────────────────────────┘
```
- 复选框填充 Action Blue 对勾
- 文字 `line-through` + `text-white/30`
- 整行降低透明度

## 5. 顶部操作栏

### 5.1 统计摘要行

```
✅ 每日待办
3 项待完成 · 1 项今天 · 0 项高优先
```

- 字体：12px Caption
- 颜色：`text-white/50`
- 高优先数字用 `#FF453A` 标红

### 5.2 过滤栏

```
[今天] [待完成] [已完成] [归档] [全部]
```

- 高 32px
- 未选中：`bg-white/5 text-white/60`
- 选中：`bg-white/15 text-white border border-white/20`
- 圆角：16px（胶囊）
- 间距：4px

### 5.3 搜索框

```
┌────────────────────────────────────────┐
│ 🔍  搜索待办...                    ✕   │
└────────────────────────────────────────┘
```

- 高 36px
- 背景：`bg-white/5`
- 描边：`0.5px rgba(255,255,255,0.08)`
- 圆角：8px
- placeholder：`text-white/30`
- 聚焦：描边变为 Action Blue

## 6. 视图切换

列表视图（默认）和看板视图通过右上角图标切换：

```
[☰ 列表] [▦ 看板]
```

看板模式：三列全宽（待处理 / 进行中 / 已完成）

## 7. 新建待办

### 7.1 快速输入（折叠态）

```
┌────────────────────────────────────────┐
│  +  添加待办...                        │
└────────────────────────────────────────┘
```
一条线，高 36px，点击展开

### 7.2 展开态

```
┌────────────────────────────────────────┐
│  [输入标题...]                          │
│  📂 分类  ⚡ 优先级  📅 截止日期       │
│                              [添加]    │
└────────────────────────────────────────┘
```
- Enter 提交
- Esc 折叠

## 9. 交互

| 交互 | 效果 |
|------|------|
| 点击复选框 | 完成/取消完成，0 延迟 |
| 点击待办行 | 打开编辑弹窗（TodoEditModal） |
| 右滑 | 归档（后续实现） |
| 拖拽排序 | @dnd-kit（已有） |
| 过滤切换 | 列表淡入淡出 200ms |
