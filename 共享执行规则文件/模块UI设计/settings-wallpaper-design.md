## 一、设置模块

### 1. 设计目标

管理全局配置。设计原则：
- **分组清晰**：每个设置区独立卡片
- **即时生效**：不需要保存按钮
- **不打扰**：设置页低频访问，保持简洁

│  │ 壁纸 ☑                             │  │

### 3. 卡片规格

| 属性 | 值 |
|------|-----|
| 宽度 | `max-w-2xl`（672px，比 720 更紧凑） |
| 卡片间距 | `space-y-4`（16px） |
| 卡片内边距 | `p-5`（20px） |
| 卡片圆角 | 12px |
| 标题 | 15px Semibold + 16px emoji |

### 4. Toggle 开关

```
欢迎  [════●]    ← 开：Action Blue 背景 + 白色圆点
日记  [●════]    ← 关：bg-white/20 + 白色圆点
```

- 宽 36px，高 20px
- 圆角 10px（pill）
- 过渡 150ms

## 二、壁纸模块

### 1. 设计目标

预览和切换壁纸。设计原则：
- **所见即所得**：缩略图实时预览
- **多种来源**：预设 / URL / 本地上传 / 纯色 / 渐变

### 2. 布局草图

```
┌──────────────────────────────────────────┐
│  🖼  壁纸                                 │
│                                          │
│  ┌────────────────────────────────────┐  │
│  │ [预设] [URL] [上传] [纯色] [渐变]   │  │ ← Tab 切换
│  └────────────────────────────────────┘  │
│                                          │
│  预设壁纸网格（3 列）：                   │
│  ┌──────┐ ┌──────┐ ┌──────┐            │
│  │ 极光  │ │ 山脉  │ │ 海洋  │            │
│  └──────┘ └──────┘ └──────┘            │
│  ...                                     │
│                                          │
│  ──────────────────────────────────────  │
│  历史记录（单行横向滚动）：               │
│  [极光] [山脉] [海洋] [自定义] ...       │
└──────────────────────────────────────────┘
```

### 3. 预设缩略图

| 属性 | 值 |
|------|-----|
| 网格列数 | 3 列（≥1024px）/ 2 列（<1024px） |
| 缩略图宽高比 | `aspect-video`（16:9） |
| 圆角 | 8px |
| 选中态 | 2px Action Blue 边框 + 右下角 `✓` |
| hover | `scale(1.02)` |

### 4. 历史记录

- 水平滚动条
- 每项 64×36px 缩略图
- 点击设为当前壁纸
- 右键删除（或 hover 显示删除按钮）

### 5. 本地上传

```
┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
│  📁  点击或拖拽上传图片          │ ← 虚线边框区域
│     支持 JPG/PNG/WebP           │
└ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
```

## 三、通用交互

| 元素 | hover | active |
|------|-------|--------|
| 卡片 | — | — |
| 按钮 | bg 加深 10% | `scale(0.98)` |
| Toggle | — | 滑动过渡 |
| 缩略图 | `scale(1.02)` + 边框亮起 | — |
