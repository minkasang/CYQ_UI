## 范式 3：纯内容 + 浮动控件（默认）

设计灵感：Apple Photos、Safari — 内容占满全屏，chrome 最小化，控件浮动在底部。

### 页面骨架

```
┌─────────────────────────────────────────────────────┐
│  2026年7月18日 星期五                    🔍 搜索...  │ ← 顶栏 40px，极简透明
│─────────────────────────────────────────────────────│
│                                                     │
│                                                     │
│                  全屏内容区                           │
│                （无固定最大宽度）                      │
│                  px-8 py-6                           │
│                                                     │
│                                                     │
│                                                     │
│─────────────────────────────────────────────────────│
│  ┌─────────────────────────────────────────────┐    │
│  │  🏠首页  ✅待办  📖日记  💬AI  ⚙设置         │    │ ← 底部浮动 Dock
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

### 关键设计要素

**顶栏（40px，透明渐变）**
- 仅显示日期（左）和搜索图标（右）
- 背景：从透明渐变到半透明（下滑时显现）
- 无任何操作按钮——操作在页面内部或底部 Dock

**底部浮动 Dock**
- 5 个图标按钮：首页、待办、日记、AI（聊天+总结合并）、设置
- 居中排列，圆角胶囊形容器
- 背景：vibrancy blur 半透明
- 当前页图标有 Action Blue 圆点标记
- 始终可见，浮动于内容之上
- 高约 64px（含 padding）

**内容区**
- 无最大宽度限制——页面自行决定
- 水平 padding：32px
- 不需要侧边栏——导航在底部
- 每个页面可以自由设计自己的内部布局

**页面如何适应**
- 待办：可做宽列表或看板（空间大了）
- 日记：可用全宽做 Markdown 编辑器 + 预览
- AI 聊天：全宽消息流
- 首页：可做大卡片或 Widget 仪表盘
- 设置：单列居中，保持可读性

### 间距体系

| Token | 值 | 用途 |
|-------|-----|------|
| 顶栏高 | 40px | 日期+搜索 |
| Dock 高 | 64px | 底部导航 |
| 内容 padding | 32px | px-8 |
| 内容垂直 padding | 24px | py-6 |
| Dock 圆角 | 20px | rounded-[20px] |

## 范式 4：Widget 仪表盘（可选布局）

首页以卡片网格呈现，每张卡片是一个模块的摘要视图。点击进入对应模块独立页面。

### 仪表盘首页骨架

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              个人工作台    v0.1                       │
│              2026年7月18日 星期五                     │
│                                                     │
│  ┌──────────────┐  ┌──────────────┐                 │
│  │  ✅ 待办      │  │  📖 日记      │                │
│  │  3 项待完成   │  │  最近：...    │                │
│  │  今天 2 项    │  │  连续 5 天    │                │
│  └──────────────┘  └──────────────┘                 │
│  ┌──────────────┐  ┌──────────────┐                 │
│  │  💬 AI 聊天   │  │  📊 数据      │                │
│  │  最近对话...   │  │  情绪 / 统计   │               │
│  └──────────────┘  └──────────────┘                 │
│                                                     │
│─────────────────────────────────────────────────────│
│  ┌─────────────────────────────────────────────┐    │
│  │  🏠首页  ✅待办  📖日记  💬AI  ⚙设置         │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

### 仪表盘特色
- 无传统导航——导航在卡片上（点卡片进入模块）
- 底部 Dock 保留（切换到其他模块的独立页面）
- 卡片：2×2 网格，每张 140-200px 高
- 卡片带实时数据摘要
- 模块独立页面仍然是纯内容范式（顶部返回+Dock）

## 色彩体系（暗色）

| Token | 值 | 用途 |
|-------|-----|------|
| 最深背景 | `#0d0d0d` | 内容区底色 |
| Dock 背景 | `rgba(30,30,32,0.85)` + blur | 底部导航 |
| 顶栏 | `rgba(0,0,0,0)` → `rgba(0,0,0,0.6)` | 渐变 |
| 卡片 | `rgba(255,255,255,0.06)` | 面板 |
| Action Blue | `#2997ff` | 唯一交互色 |

## 与 DefaultLayout 差异

| 维度 | DefaultLayout | macOS v3 |
|------|--------------|----------|
| 导航位置 | 左侧垂直侧边栏 | 底部水平 Dock |
| 导航可见性 | 始终展开 | 始终可见，浮动 |
| 内容宽度 | max-w-6xl | 无限制 |
| 顶栏 | 日期+操作按钮 | 仅日期+搜索 |
| 整体感觉 | 传统 Web App | 原生 macOS 沉浸式 |
