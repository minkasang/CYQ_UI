# 任务拆解 - 布局系统

## 任务概述

创建可切换、可导入/导出的布局模块系统。每个布局控制整体页面结构（侧边栏位置、顶栏样式、内容宽度、间距）。

---

## 步骤拆解

### 步骤 1：类型定义 + Store

| 子步骤 | 任务 | 验证方式 |
|--------|------|----------|
| 1.1 | 创建 `types/layout.ts`，定义 LayoutConfig 接口 | tsc |
| 1.2 | 创建 `useLayoutStore.ts`（Zustand persist + 4个内置布局） | tsc |

### 步骤 2：LayoutManager 组件

| 子步骤 | 任务 | 验证方式 |
|--------|------|----------|
| 2.1 | 创建切换 UI（显示所有布局，点选切换） | 手动 |
| 2.2 | 导入 JSON / 导出 JSON 功能 | 手动 |
| 2.3 | 删除自定义布局功能 | 手动 |

### 步骤 3：接入现有组件

| 子步骤 | 任务 | 验证方式 |
|--------|------|----------|
| 3.1 | Sidebar 接入 sidebarPosition / sidebarWidth | 手动 |
| 3.2 | TopBar 接入 topBarStyle | 手动 |
| 3.3 | Layout 接入 contentMaxWidth / contentPadding | 手动 |

### 步骤 4：集成到设置页面

| 子步骤 | 任务 | 验证方式 |
|--------|------|----------|
| 4.1 | SettingsPage 或独立入口嵌入 LayoutManager | 手动 |
| 4.2 | 切换布局 → 即时生效 | 手动 |
