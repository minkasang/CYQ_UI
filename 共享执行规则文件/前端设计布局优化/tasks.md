# 任务拆解 - 前端设计布局优化

## 任务概述

以 Apple 设计语言（macos-design + apple-design）为参考，逐步优化个人工作台的 UI 布局和视觉一致性，打造类似 Apple 系统风格的排版。

## 设计参考

| Skill | 内容 |
|-------|------|
| macos-design | macOS 原生 App 模式：布局、交互、动画 |
| apple-design | Apple 设计 token：SF Pro 字体、Action Blue 色彩、组件规范 |
| Apple DESIGN.md | 完整 Apple 设计规范（必要时读取原文） |

## 阶段计划

### 阶段 1：安装设计参考 skill
- 安装 Apple、Linear、Notion 三个 DESIGN.md 为 Reasonix skill
- 验证 skill 可被 AI 调用
- **状态**：in_progress

### 阶段 2：设计审计
- 对照各设计规范，列出当前工作台的 UI 问题清单
- 输出 `architecture.md`

### 阶段 3：逐步优化
- 按优先级逐模块优化
- 每步产出 + 验证
