# 任务拆解 - 每日灵感

## 任务概述

新建 `inspiration` 模块：个人数字 Commonplace Book，捕获触动自己的名言/哲理/好句子，支持随机回顾以防止遗忘。

## 设计参考

| 来源 | 借鉴 |
|------|------|
| Readwise Daily Review | 单条大卡片展示，一次只看一条，优先回顾遗忘的 |
| Obsidian 标签系统 | 多标签替代文件夹分类，标签互联 |
| 洛克/Luhmann 方法论 | 永久ID + 多轴归类 + 三层笔记 |
