# 更新日志

## v0.1.0 (2026-06-15) - MVP

### 🎉 首次发布

个人工作台首个可发布版本。

### ✨ 功能

- **基础布局**
  - 侧边栏导航（首页/待办/日记/AI/壁纸/设置）
  - 顶部栏（日期、导出、导入、玻璃调参）
  - 全局背景（图片/渐变/纯色）

- **流体玻璃效果**
  - 集成 liquid-glass-react
  - 6 个可调参数
  - 4 种变形模式（standard/polar/prominent/shader）
  - 实时调参面板

- **每日待办**
  - 增删改查
  - 4 种分类（工作/生活/学习/其他）
  - 3 种优先级（高/中/低）
  - 截止日期
  - 4 种过滤（今天/待完成/已完成/全部）
  - 统计卡片

- **日记**
  - Markdown 编辑器
  - 实时预览
  - 自动保存（800ms 防抖）
  - 按日期组织

- **AI 总结**
  - 支持 6 种 AI 服务（DeepSeek/OpenAI/Claude/Kimi/智谱/自定义）
  - 总结日记
  - 总结长文本
  - 流式输出
  - 连接测试

- **壁纸管理**
  - 8 种预设渐变
  - 网络图片 URL
  - 本地图片上传
  - 历史记录（最多 20 张）
  - 一键切换

- **数据管理**
  - 导出全部数据（JSON）
  - 导入数据
  - 清空所有数据（二次确认）

### 🔧 技术栈

- React 18.3.1
- Vite 5.3
- TypeScript 5.5
- Tailwind CSS 3.4
- Zustand 4.5
- React Router 6.26
- liquid-glass-react 1.1.1
- react-markdown 9.0
