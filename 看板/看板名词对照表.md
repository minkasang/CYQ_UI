## 液态玻璃效果

| 你说的 | 技术名称 | 位置/说明 |
|--------|----------|-----------|
| 毛玻璃效果、玻璃特效 | **液态玻璃 (liquid-glass)** | WebGL + GLSL 渲染的折射/模糊效果 |
| 玻璃的参数调节 | **调参面板** | 16 个滑块 + 2 个开关 + 恢复默认按钮 |
| 玻璃里看到背景透过来 | **折射 (refraction)** | 参数名 `refraction`，默认 0.69 |
| 玻璃边缘的彩虹边 | **色散/色差 (chromAberration)** | 参数名 `chromAberration`，默认 0.05 |
| 玻璃边缘的发光 | **边缘高光 (edgeHighlight)** | 参数名 `edgeHighlight` |
| 玻璃镜面反光 | **镜面反射 (specular)** | 参数名 `specular` |
| 玻璃边角更亮的效果 | **菲涅尔 (fresnel)** | 参数名 `fresnel` |
| 玻璃圆角大小 | **圆角 (cornerRadius)** | 参数名 `cornerRadius`，默认 65 |
| 玻璃变形扭曲 | **扭曲 (distortion)** | 参数名 `distortion` |
| 玻璃下方那两层壁纸 | **bgCanvas 采样层** | 离屏画布，给 WebGL 读取壁纸像素用的 |
| 壁纸显示错位/下面有白条 | **壁纸偏移 (sourceY offset)** | `sourceY = rect.top - 20` 里的 -20 |

## 调参面板参数速查

| 参数名 | 中文 | 默认值 | 范围 |
|--------|------|--------|------|
| blurAmount | 模糊强度 | 0 | 0-30 |
| refraction | 折射强度 | 0.69 | 0-2 |
| chromAberration | 色散 | 0.05 | 0-0.5 |
| edgeHighlight | 边缘高光 | 0.05 | 0-1 |
| specular | 镜面反射 | 0 | 0-2 |
| fresnel | 菲涅尔 | 1 | 0-3 |
| distortion | 扭曲 | 0 | 0-1 |
| cornerRadius | 圆角 | 65 | 0-100 |
| zRadius | 深度半径 | 40 | 0-100 |
| opacity | 不透明度 | 1 | 0-1 |
| saturation | 饱和度 | 0 | -1~1 |
| tintStrength | 色调强度 | 0 | 0-1 |
| brightness | 亮度 | 0 | -1~1 |
| shadowOpacity | 阴影不透明度 | 0.3 | 0-1 |
| shadowSpread | 阴影扩散 | 10 | 0-40 |
| shadowOffsetY | 阴影Y偏移 | 1 | -20~20 |
| floating | 浮动模式 | false | 开关 |
| button | 按钮模式 | false | 开关 |
| bevelMode | 斜面模式 | 0 | 0/1 |

## 如何向 AI 描述你看版上的问题

**举例：**
- "左边那一大块规范列表里，那个感知系统规范右边有个红色字" → AI 查表知道你说的是 `spec-item` 里的 `风险标记`
- "快捷入口下面那些小卡片" → AI 知道是 `spec-item`（规范条目）
- "右边弹出来的那个看文档的窗口" → AI 知道是 `preview-panel`
- "毛玻璃下面有条白线" → AI 知道是液态玻璃的 `壁纸偏移` 问题
- "那个调参的东西" → AI 知道是 `glass-ctrl-panel`
