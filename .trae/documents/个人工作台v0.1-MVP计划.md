## 1. 计划摘要（Summary）

构建一个基于 **React + Vite + 流体玻璃效果** 的个人工作台 Web 应用，核心目的是**督促个人习惯、安排工作与生活、记录思考**。

**v0.1 MVP 范围**（先做能用的最小版本）：
1. 基础布局（侧边栏 + 主内容区 + 顶部栏）
2. 流体玻璃效果（使用 liquid-glass-react 库）
3. 每日待办（Todo 增删改查）
4. 日记（Markdown 编辑器）
5. AI 总结（DeepSeek API + 一键总结日记）
6. 壁纸管理（网络图片 + 本地上传）

**v0.2+ 后续阶段**：警醒墙（弹窗+列表+定时）、灵感记录、文件连接、多模型切换、多设备同步等。

### 3.3 关键设计决策

#### 决策 1：玻璃效果默认参数
- 默认 `mode="standard"`（标准变形，兼容性最好）
- 默认 `blurAmount=0.4`、`saturation=140`（视觉效果好但不过分）
- 默认 `cornerRadius=16`（现代感）
- 用户可在设置页调整，存 localStorage

#### 决策 2：AI 服务架构
```
┌──────────────────┐
│   AISummary UI   │  (组件层)
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│   aiService.ts   │  (抽象层 - 接口统一)
└────────┬─────────┘
         │
   ┌─────┴──────┬──────────┐
   ▼            ▼          ▼
┌────────┐  ┌────────┐  ┌────────┐
│DeepSeek│  │ OpenAI │  │  其他  │
│Adapter │  │Adapter │  │Adapter │
└────────┘  └────────┘  └────────┘
```

#### 决策 3：壁纸 + 玻璃的协调
- 壁纸存 IndexedDB（支持大文件）
- 每次壁纸变更时，触发 LiquidGlass 组件的 `key` 变化，强制重新渲染滤镜
- 提供"重置玻璃"按钮兜底

#### 决策 4：数据存储策略
- v0.1 完全本地：localStorage + IndexedDB
- 提供"导出全部数据"按钮，生成 `.json` 文件
- 提供"导入数据"按钮，恢复 JSON
- 未来 v0.3+ 再考虑云同步（Supabase）

## 4. 假设与决策（Assumptions & Decisions）

### 4.1 用户已确认的决策
| 决策点 | 用户选择 |
|--------|----------|
| 技术栈 | React + Vite（基于"第一种方案"的确认） |
| 玻璃效果 | 流体玻璃（liquid-glass-react） |
| AI 服务 | DeepSeek + 可扩展其他（用户提及） |
| 多设备同步 | 本地优先（v0.1），v0.3+ 再考虑云 |
| 开发方式 | 逐步实现 MVP |
| 弹窗/警醒 | 用户在 v0.2+ 实现（v0.1 不含） |

### 4.2 AI 自主决定（用户未明确表态，但基于项目规范推断）
| 决策点 | AI 决定 | 理由 |
|--------|---------|------|
| 命名 | 看板效果叫"光学玻璃"、新效果叫"流体玻璃" | 基于技术原理分析 |
| 数据存储 | localStorage + IndexedDB | 用户选本地优先 |
| 样式方案 | Tailwind CSS | 现代化、快、用户无反对 |
| API Key 存储 | localStorage（不加密） | 简化开发，未来可加密码锁 |
| 自动保存 | 实时保存到 localStorage | 防止数据丢失 |
| 响应式 | v0.1 仅做桌面端（≥1024px） | 个人电脑用，不浪费精力 |

### 4.3 风险与应对
| 风险 | 应对策略 |
|------|----------|
| liquid-glass-react 在某些浏览器降级 | 已分析支持情况，自动降级为 backdrop-filter |
| DeepSeek API 限流或欠费 | 显示友好错误，提示用户检查 API Key |
| 用户没配 API Key | AI 功能入口禁用并显示配置引导 |
| localStorage 容量限制（5-10MB） | 日记/图片用 IndexedDB，文本用 localStorage |
| 壁纸加载慢 | 预加载 + 模糊占位 |
