# 用户画像 (User Profile)

> AI 在与用户协作中总结的偏好。每次对话开始前应读取。

---

## UI 风格偏好

- **喜欢**：macOS 原生风格、毛玻璃 + 液态玻璃组合、暗色背景、大片留白
- **不喜欢**：独立网页跳转（偏好首页内滚动）、模板感 AI 设计、同一卡片内混两种玻璃效果并排
- **常用模式**：Dock 导航、Scroll Snap 分屏、毛玻璃卡片包在液态玻璃面板内

## 交互偏好

- **喜欢**：自然滚动（非强制吸附）、点击导航滚到对应位置
- **不喜欢**：`snap-mandatory` 强制跳转、点击模块跳独立页面
- **习惯**：用底部 Dock 导航、侧边栏跳转 section

## 代码组织偏好

- 模块化架构（ModuleManager + 热拔插）
- 每个模块独立目录（`modules/{name}/`）
- Zustand persist 存储
- Tailwind CSS 优先（不用 inline style）

## 术语习惯

- "模块" 不是 "功能页"
- "壳" = 布局壳（Layout Shell）
- "灵感" = Commonplace Book
- 语音输入 → 可能有繁体/错字 → 自动纠正

## 沟通风格

- 直接、简洁
- 一次只说一件事
- 错了会直接指出
- 喜欢先问需求再做
