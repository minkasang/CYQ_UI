# 项目启动命令

> ⚠️ **重要**：所有命令都需要在项目根目录 `cyq_UI/` 下执行！
>
> 如果你当前在其他目录（如 `看板/` 或 `个人工作台/`），先回到根目录：
> ```bash
> cd /media/cyq/455E28568C7503F7/project/cyq_UI
> ```

## 1. 后端服务（server.py）

提供文件读写 API，端口 8090。位于项目根目录。

```bash
python server.py &
```

## 2. 前端服务（个人工作台）

React + Vite 开发服务器，端口 5174。

```bash
cd 个人工作台 && npx vite
```

## 3. 一键启动（推荐）

无论在哪个目录，一行命令启动，Ctrl+C 同时停止所有服务：

```bash
/media/cyq/455E28568C7503F7/project/cyq_UI/start.sh
```

或添加别名（可选）：

```bash
# 添加到 ~/.bashrc
alias start-pw='/media/cyq/455E28568C7503F7/project/cyq_UI/start.sh'

# 之后直接用
start-pw
```

## 4. 清理命令

清理 npm 缓存：

```bash
npm cache clean --force
```

## 5. 数据存储位置

数据文件存储在 `个人工作台/data/` 目录：
- `todos.json` - 待办数据
- `diaries.json` - 日记数据
- `ai_config.json` - AI 配置
- `settings.json` - 设置
- `wallpaper.json` - 壁纸
- `glass_params.json` - 玻璃参数
