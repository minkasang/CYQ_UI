# 项目启动命令

## 1. 后端服务（server.py）

提供文件读写 API，端口 8090。

```bash
cd 看板 && python server.py
```

启动后可用 API：
- `GET /api/read?path=xxx` - 读取文件
- `POST /api/write` - 写入文件
- `GET /api/list?dir=xxx` - 列目录
- `GET /api/search?q=xxx` - 搜索文件内容

## 2. 前端服务（个人工作台）

React + Vite 开发服务器，端口 5174。

```bash
cd 个人工作台 && npx vite
```

如果端口被占用，Vite 会自动尝试其他端口。

## 3. 一键启动（两个服务）

```bash
# 先启动后端
cd 看板 && python server.py &

# 再启动前端
cd 个人工作台 && npx vite
```

## 4. 清理命令

清理 npm 缓存：

```bash
npm cache clean --force
```

## 5. 数据存储位置

数据文件存储在 `个人工作台/data/` 目录：
- `todos.json` - 待办数据
- `diaries.json` - 日记数据
- `ai_config.json` - AI 配置
- `settings.json` - 设置
- `wallpaper.json` - 壁纸
- `glass_params.json` - 玻璃参数
