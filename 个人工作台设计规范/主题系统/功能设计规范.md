# 功能设计规范 — 主题系统

> 规范类型：功能 + 用户路径 + 视觉 + 状态覆盖
> 参考：architecture.md

---

## 功能清单

### 已规划

| 功能 | 说明 |
|------|------|
| 主题浏览 | 左侧列表展示所有主题（内置 + 自定义），选中高亮 |
| 实时预览 | 选中主题立即应用到当前页面，所见即所得 |
| 新建主题 | 选引擎 → 配参数 → 配字体 → 配壁纸 → 保存 |
| 另存为 | 从现有调参面板「保存为新主题」 |
| 参数配置 | 右侧面板根据引擎动态渲染滑块（折射率/模糊度/边框/...） |
| 字体配置 | 字体栈选择 + 基础字号 |
| 壁纸配置 | 内置壁纸选择器（复用现有） |
| 删除主题 | 自定义主题可删除，内置不可删 |
| 导入导出 | JSON 文件导入/导出主题 |
| 模块注册 | 作为模块接入 Dock、设置页模块开关 |

### 不做什么

- ❌ 不重写引擎（LiquidGlass/Flat）
- ❌ 不改变现有调参面板（GlassControlPanel 保留）
- ❌ 不搞参数分类（通用/专属）

## 用户路径

### 首次使用
```
打开设置/主题页 → 看到 2-3 个内置主题 → 点「默认玻璃」→ 页面立即应用
→ 右侧显示玻璃参数滑块 → 调了折射率 → 点「保存为新主题」→ 命名「我的玻璃」
→ 列表出现「我的玻璃」→ 以后可快速切换
```

### 日常使用
```
Dock 点🎨主题 → 进入主题页面 → 列表选一个主题 → 立即预览
→ 不满意换另一个 → 满意了关掉页面
```

### 创建自定义主题
```
主题页面点「+ 新建」→ 弹窗选引擎（液态玻璃/扁平）→ 进入配置
→ 调参数 → 选字体 → 选壁纸 → 命名 → 保存
→ 列表出现新主题
```

## 状态覆盖

### 空状态（无自定义主题）
- 列表显示 2-3 个内置主题
- 底部「+ 新建」按钮可见

### 正常态（有内置 + 自定义）
- 列表分两组：内置 / 自定义（带分隔线）
- 内置主题右侧有 🔒 标记
- 自定义主题右侧有 🗑 删除按钮

### 选中态
- 列表项高亮（accent 色左边框）
- 中间页面立即应用主题
- 右侧显示对应引擎的参数面板

### 编辑态
- 调参数时实时预览变化
- 参数修改后「保存为新主题」按钮亮起
- 点保存 → 弹命名框 → 确认 → 列表新增

### 导入失败
- JSON 格式不正确 → Toast「无效的主题文件」
- 引擎不存在 → Toast「引擎 xxx 不可用，已跳过」

### 删除确认
- 点 🗑 →「确定删除「xxx」主题？」→ 确认 → 从列表移除 + 文件更新
