# 功能设计规范 - 每日灵感

> 个人数字 Commonplace Book 模块
> 规范类型：功能 + 用户路径 + 视觉 + 状态覆盖

---

## 功能清单

### 已实现（Batch 1+2）

| 功能 | 说明 |
|------|------|
| 快速捕获 | 输入内容 + 来源 + 交互式标签输入（自动补全+颜色chip），回车即存 |
| 标签系统 | 自定义标签名 + 创建时 10 色自动分配 + 输入时下拉匹配已有标签 + 颜色编辑 + 使用频率统计 + 删除无用标签 |
| 随机回顾 | 首页展示一条，优先推荐最久没看的，触动程度加权 |
| 触动程度 | 1-3 星，一键切换 |
| 收藏标记 | ❤ 一键收藏/取消 |
| 换一条 | 标记当前已回顾 → 算法推荐下一条 |
| 最近列表 | 右侧展示全部记录，可滚动（最大 320px） |
| 行内编辑 | ✏️ 展开编辑表单，修改内容/来源/标签/感想 |
| 删除确认 | 🗑 → confirm 弹窗 → 删除 |
| 搜索过滤 | 搜索内容/来源/标签，实时过滤 |
| 标签筛选 | 点标签 chip 筛选，再点取消 |
| 排序切换 | 最新 / 最早 / 触动高→低 / 触动低→高 |
| 双卡布局 | 桌面端左右并排，移动端上下堆叠 |
| 玻璃效果 | 外层液态玻璃 + 内层双毛玻璃卡片 |
| 5 壳导航 | Sidebar / Dock / TopNav 均可导航到首页灵感 section |
| 模块开关 | SettingsPage 自动出现开关，可关闭 |
| Scroll Snap | 灵感 section 至少占一屏，proximity 吸附 |

## 用户路径

### 首次使用（空状态）
```
打开首页 → 滚到灵感 section → 看到 "记下第一句触动你的话" → 点 [写一句]
→ 输入内容 → 回车/保存 → 回顾卡片出现 → 列表出现这一条
```

### 日常使用
```
滚到灵感 section → 回顾卡片随机展示一条 → ⭐调触动 → ❤收藏 → [换一条]
→ 右侧列表可搜索/标签筛选/排序 → 点✏️行内编辑 → 点🗑删除（确认）
```

### Dock 导航
```
点 Dock 💡灵感 → 首页滚到 #inspiration → 吸住
```

## 状态覆盖

### 空状态（0 条记录）
- 显示 "记下第一句触动你的话"
- [写一句] 按钮
- 右侧列表隐藏

### 正常态（≥1 条）
- 回顾卡片展示一条随机记录
- 右侧列表显示筛选后的全部记录
- 每条可编辑/删除

### 搜索无结果
- 列表区显示 "没有匹配的记录"

### 编辑态
- 列表项展开为行内编辑表单
- ✗ 取消 / ✓ 保存

### 删除确认
- 点🗑 → 浏览器 confirm 弹窗 → 确认后删除
