# 功能设计规范 - 人生图谱

> 从“每日灵感”升级而来。  
> 定位：记录那些会慢慢改变用户的东西。

---

## 功能清单

### 第一阶段要做（实现状态：✅ 已全部落地）

- 低门槛收集箱：内容必填，其他字段都可选。
- 多类型记录：碎片、摘录、洞察、原则、模式、行动。
- 多维度标记：价值观、关系、工作、情绪、身体、学习、创造、原则等。
- 漫游模式：像翻阅私人内在空间一样回看记录。
- 锚定模式：把某条洞察沉淀为原则提醒或小行动实验。
- 内在图谱概览：根据维度和标签展示反复主题。
- 原则提醒：集中展示重要原则。
- 小行动实验：记录触发条件、行动和状态。
- 保留旧能力：搜索、标签、来源、感想、收藏、触动程度、编辑、删除、换一条。

### 第一阶段不做

- 不自动读取日记。
- 不调用 AI 自动分析。
- 不做强制打卡。
- 不做社交分享。
- 不做跨设备同步。

---

## 用户路径

### 随手记录

```text
打开人生图谱
  → 在收集箱写下想法
  → 可选填写来源、标签、类型、维度、感想
  → 保存
  → 记录进入漫游流和内在图谱统计
```

### 提炼洞察

```text
看到一条记录
  → 点击编辑
  → 补充“它提醒我什么”
  → 可选写成“原则提醒”
  → 保存
```

### 转化行动

```text
某条洞察很重要
  → 写一个小行动实验
  → 填写“如果遇到什么情况”
  → 填写“我就做什么”
  → 后续标记进行中或已完成
```

### 安静回看

```text
低能量时打开
  → 看今日回看卡
  → 浏览原则提醒与旧记录
  → 不被强制要求行动
```

## 状态覆盖

- 空状态：显示“先不用整理，写下第一件想留下的东西”。
- 正常态：显示回看卡、收集箱、图谱概览、记录流。
- 搜索无结果：提示“没有匹配的记录”。
- 无原则：显示“重要原则会在这里慢慢长出来”。
- 无行动实验：显示“准备改变时，再给自己一个很小的实验”。
- 编辑态：展开完整字段，允许取消或保存。
- 旧数据：按碎片记录显示，缺少新增字段时不显示对应区块。

## 视觉参考

- 主氛围：私人、安静、精神性的内在空间。
- 交互气质：默认不催促，必要时展开理性方法论。
- 版式参考：Editorial Grid / Magazine，用大字号引用、留白、分区标题和不对称网格建立排版感。
- 项目风格：延续 macOS + 液态玻璃 + 暗色背景，但减少“工具箱堆按钮”的感觉。
